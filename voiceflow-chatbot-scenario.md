## 📊 챗봇 구조 한눈에 보기

```
[시작]
   ↓
[환영 인사 + 메인 메뉴]
   ↓
├─➊ 서비스 소개
│   ├─ AI 마케팅 자동화
│   ├─ 데이터 분석
│   ├─ 타겟 고객 발굴
│   └─ [상담 신청으로 연결]
│
├─➋ 가격 문의
│   ├─ 스타터 플랜 (99만원)
│   ├─ 프로 플랜 (299만원) ⭐추천
│   ├─ 엔터프라이즈 (맞춤 견적)
│   └─ [상담 신청으로 연결]
│
├─➌ 성공 사례
│   ├─ 이커머스 사례
│   ├─ SaaS 사례
│   ├─ 교육 사례
│   └─ [상담 신청으로 연결]
│
├─➍ 상담 신청 ⭐핵심
│   ├─ 회사명 수집
│   ├─ 담당자명 수집
│   ├─ 이메일 수집
│   ├─ 전화번호 수집
│   ├─ 관심 서비스 수집
│   ├─ [데이터 저장/전송]
│   └─ [완료 메시지]
│
└─➎ FAQ
    ├─ 계약 기간
    ├─ 적합 업종
    ├─ 설정 시간
    ├─ 데이터 보안
    └─ 환불 정책
```

## 🎬 Step 1: Voiceflow 프로젝트 생성

### 1.1 계정 설정
1. https://www.voiceflow.com 접속
2. "Start for Free" 클릭
3. Google 계정으로 간편 가입
4. Workspace 이름: "MarketingPro"

### 1.2 프로젝트 생성
1. "Create Project" 버튼 클릭
2. Template: **"Start from Scratch"** 선택
3. Project Name: `마케팅 리드 수집 챗봇`
4. Assistant Type: **"Chat Assistant"** 선택
5. Create 클릭

### 6.8 데이터 전송 API 블록 ⚙️

**블록 타입**: API

**설정**:

**Method**: POST

**URL**:
```
https://hooks.zapier.com/hooks/catch/YOUR_WEBHOOK_ID/
```
또는
```
https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec
```
또는 직접 백엔드
```
https://your-backend.com/api/leads
```

**Headers**:
```json
{
  "Content-Type": "application/json"
}
```

**Body**:
```json
{
  "company": "{company}",
  "name": "{name}",
  "email": "{email}",
  "phone": "{phone}",
  "service": "{service}",
  "budget": "{budget}",
  "source": "chatbot",
  "timestamp": "{timestamp}",
  "page_url": "{page_url}"
}
```

**Response Mapping**:
- `success` → `{api_success}`
- `lead_id` → `{lead_id}`

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);

  sheet.appendRow([
    new Date(),
    data.company,
    data.name,
    data.email,
    data.phone,
    data.service,
    data.budget
  ]);

  return ContentService.createTextOutput(JSON.stringify({
    success: true,
    lead_id: sheet.getLastRow()
  })).setMimeType(ContentService.MimeType.JSON);
