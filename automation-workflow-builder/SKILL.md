# ⚡ 자동화 워크플로우 빌더 스킬

이 스킬은 BSD 바이브코딩 수강생들이 **반복적인 업무를 자동화**하여 생산성을 극대화할 수 있도록 돕습니다.

## 📋 이 스킬이 하는 일

1. **업무 분석**: 자동화 가능한 반복 작업 식별
2. **워크플로우 설계**: 단계별 자동화 프로세스 구성
3. **도구 추천**: Zapier, Make, n8n 중 최적 선택
4. **구현 가이드**: 상세한 설정 방법 제공
5. **에러 처리**: 예외 상황 대응 방법 포함

## 🎯 언제 이 스킬을 사용하나요?

- "이메일 오면 자동으로 스프레드시트에 저장하고 싶어요"
- "폼 작성하면 슬랙으로 알림 받고 싶어요"
- "인스타그램 게시물 자동으로 여러 플랫폼에 올리고 싶어요"
- "매일 아침 어제 판매 데이터를 요약해서 보내주세요"

## 🛠️ 지원 플랫폼

### Zapier (초보자 추천) ⭐
**장점**:
- 5,000+ 앱 연동
- 직관적인 UI
- 풍부한 템플릿
- 안정적인 운영

**단점**:
- 유료 플랜 필요 (월 $19.99~)
- 복잡한 로직 제한적

**추천 대상**: 
- 빠르게 시작하고 싶은 초보자
- 예산이 있는 비즈니스 오너
- 안정성이 중요한 업무

### Make (구 Integromat) ⭐⭐
**장점**:
- 시각적 플로우 차트
- 복잡한 시나리오 가능
- 무료 플랜 제공 (월 1,000 작업)
- 조건부 로직 쉬움

**단점**:
- 학습 곡선 있음
- 연동 앱 수는 Zapier보다 적음

**추천 대상**:
- 중급 이상 사용자
- 복잡한 자동화 필요
- 비용 절감 원하는 경우

### n8n (개발자/파워유저 추천) ⭐⭐⭐
**장점**:
- 오픈소스 (무료)
- 자체 호스팅 가능
- 무제한 워크플로우
- JavaScript 커스터마이징

**단점**:
- 기술적 지식 필요
- 서버 설정 필요

**추천 대상**:
- 개발 지식 있는 사용자
- 완전한 커스터마이징 원함
- 데이터 보안이 중요한 경우

## 💡 자주 사용되는 자동화 시나리오

### 1. 이메일 자동화
```
시나리오: 고객 문의 이메일 → 자동 분류 → 담당자에게 할당

플로우:
1. Gmail: 새 이메일 수신
2. ChatGPT: 이메일 내용 분석 및 카테고리 분류
3. Notion: 문의 데이터베이스에 저장
4. Slack: 담당자에게 멘션하여 알림
5. Gmail: 자동 응답 이메일 발송
```

### 2. 소셜미디어 자동 게시
```
시나리오: 하나의 콘텐츠 → 여러 플랫폼 동시 게시

플로우:
1. Notion: 콘텐츠 캘린더에서 게시 예정 항목 확인
2. ChatGPT: 플랫폼별 최적화된 캡션 생성
3. Instagram: 피드/릴스 게시
4. Twitter/X: 트윗 게시
5. LinkedIn: 포스트 게시
6. Notion: 게시 완료 상태 업데이트
```

### 3. 판매 데이터 리포트
```
시나리오: 매일 아침 전날 판매 요약 리포트 자동 생성

플로우:
1. Schedule: 매일 오전 9시 트리거
2. Shopify/Toss: 전날 판매 데이터 조회
3. Google Sheets: 데이터 추가
4. ChatGPT: 인사이트 분석 및 요약 생성
5. Slack/Email: 팀에게 리포트 전송
```

### 4. 웨비나/이벤트 자동화
```
시나리오: 웨비나 신청 → 자동 확인 → 리마인더 발송

플로우:
1. Typeform/Google Form: 신청서 제출
2. Google Calendar: 참가자 캘린더에 이벤트 추가
3. Notion: 참가자 데이터베이스 저장
4. Email: 확인 메일 발송 (Zoom 링크 포함)
5. [D-1 시간] Email: 리마인더 발송
6. [이벤트 후] Email: 감사 메일 + 자료 링크
```

### 5. 콘텐츠 백업 자동화
```
시나리오: 여러 플랫폼 콘텐츠 자동 백업

플로우:
1. Instagram: 새 게시물 확인
2. Google Drive: 이미지 저장
3. Notion: 게시 내용 및 통계 저장
4. Airtable: 콘텐츠 캘린더 업데이트
```

### 6. 리드 관리 자동화
```
시나리오: 랜딩페이지 리드 → CRM 저장 → 영업팀 알림

플로우:
1. Tally/Typeform: 리드 폼 제출
2. ChatGPT: 리드 스코어링 (예: 적합도 분석)
3. HubSpot/Notion: CRM에 리드 추가
4. Slack: 영업팀에 고품질 리드 알림
5. Email: 자동 웰컴 이메일 시퀀스 시작
```

```
- 대체 경로 (fallback)
```

## 🔧 플랫폼별 설정 가이드

### Zapier 기본 설정

#### Step 1: 트리거 설정
```
1. "Create Zap" 클릭
2. 트리거 앱 선택 (예: Google Forms)
3. 트리거 이벤트 선택 (예: New Response)
4. 계정 연결
5. 테스트 데이터로 확인
```

#### Step 2: 액션 추가
```
1. "+" 버튼으로 액션 추가
2. 액션 앱 선택 (예: Notion)
3. 액션 이벤트 선택 (예: Create Database Item)
4. 필드 매핑 (트리거 데이터 → 액션 필드)
5. 테스트 실행
```

#### Step 3: 조건 로직 (선택사항)
```
1. "Paths" 또는 "Filter" 추가
2. 조건 설정 (예: Email contains "@gmail.com")
3. 조건별 다른 액션 설정
```

### Make 시나리오 구축

#### 시각적 에디터 활용
```
1. "Create a new scenario" 클릭
2. 첫 모듈 추가 (트리거)
3. + 버튼으로 다음 모듈 연결
4. 각 모듈 간 데이터 매핑
5. Run once로 테스트
6. Schedule 설정 후 활성화
```

#### 고급 기능
```
- Router: 하나의 플로우를 여러 경로로 분기
- Iterator: 배열 데이터를 하나씩 처리
- Aggregator: 여러 데이터를 하나로 합침
- Filter: 조건에 맞는 데이터만 통과
- Error Handler: 에러 발생 시 처리
```

## 📊 자동화 ROI 계산

### 시간 절약 계산법
```
월간 절약 시간 = 
  (작업 1회 소요 시간) × (월간 작업 빈도)

예시:
- 이메일 분류: 5분 × 50회/월 = 250분 (4.2시간)
- SNS 게시: 15분 × 20회/월 = 300분 (5시간)
- 데이터 입력: 10분 × 100회/월 = 1,000분 (16.7시간)

총 절약: 월 26시간 = 연간 312시간
```

### 비용 대비 효과
```
자동화 비용:
- Zapier 프로페셔널: $49/월
- Make 프로: $29/월
- n8n 호스팅: $20/월

절약 가치:
- 시간당 가치 $30 기준
- 월 26시간 × $30 = $780
- 순 이익: $780 - $49 = $731/월

ROI = (731/49) × 100 = 1,492% 🎉
```

```
- 에러 핸들러로 실패 데이터 저장

```

```
- Delay/Sleep 단계 추가
- 배치 처리 대신 하나씩 처리
- 피크 타임 피해서 스케줄 설정
```

### 4. 권한 문제

```
- 정기적으로 연결 상태 확인

- OAuth 토큰 갱신 자동화
```
